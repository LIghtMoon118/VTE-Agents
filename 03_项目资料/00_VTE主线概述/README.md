# VTE 智能预警目录说明

## 目录结构

- `00_项目主页`：主线定位、临床问题、数据、模型、验证和风险边界。
- `01_资料`：指南、文献、政策、既有材料。
- `02_研究设计`：研究方案、变量表、统计计划、伦理材料。
- `03_系统方案`：产品需求、流程、功能、部署和质控方案。
- `04_论文课题`：论文框架、课题申报、返修材料。
- `05_归档`：旧版本和完成材料。
- `06_肥胖人群多模态RAG横向课题`：基于多模态大模型与 RAG 的肥胖人群 VTE 动态风险预测及精准防控横向课题。
- `07_VTE智能体`：基于既有 VTE 智能管理系统和专利基础的智能体升级方向。

## 当前重点

优先把 VTE 主线拆成三个可执行方向：

1. 科研论文方向。
2. 课题申报方向。
3. 院内系统和质量改进方向。
4. 肥胖等特定高危人群精准防控横向课题方向。
5. VTE 智能体方向。
